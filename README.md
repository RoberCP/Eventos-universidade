# UniEventos

Aplicativo para visualização e inscrição em eventos da Universidade.

## Funcionalidades

- Navegação entre telas
- Formulário de login
- Busca de dados de eventos via API

## Cookbook Utilizados

- [Fetch data from the internet](https://flutter.dev/docs/cookbook/networking/fetch-data)
- [Create a form](https://flutter.dev/docs/cookbook/forms/validation)

## Configuração do Projeto

1. Clone o repositório.
2. Execute `flutter pub get` para instalar as dependências.
3. Execute o aplicativo com `flutter run`.

## Estrutura do Projeto

- `lib/`: Contém o código fonte do aplicativo.
